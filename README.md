## 🎓 Graduation Project

📍 **Project Title:** AVIR – AI-Powered Early Detection and Awareness Platform for Breast Cancer

## 🧠 Overview

AVIR is a smart healthcare platform built to support early detection and awareness of breast cancer. The system combines a secure backend, AI-powered diagnostic tools, and an interactive chatbot to guide users through self-examination, medical image analysis, and continuous health reminders — all in one connected platform.

## 📱 Platform Features

### 🔐 Authentication
- Secure Login and Sign-Up flow.
- JWT-based authentication and authorization.
- Role-based access for users and administrators.

### 🏠 Core Modules

**1. 🩺 Self-Exam & Reminders**
- Guides users through monthly self-examination steps.
- Automated email reminders scheduled with Hangfire.
- Tracks self-exam history over time.

**2. 🖼️ Medical Image Analysis**
- Upload breast scan images for AI-assisted analysis.
- Deep learning model trained for breast cancer detection.
- Stores and displays results alongside the original image.

**3. 💬 AI Chatbot**
- Conversational assistant for breast cancer awareness and FAQs.
- Provides guidance based on user symptoms and concerns.
- Powered by an LLM-based conversational model.

**4. 📚 Radiology & Community Library**
- Educational resource library for radiology references.
- Community posts and survivor stories for support and awareness.

### 🔄 Supporting Features

| Feature | Description |
|---|---|
| 📋 Symptom Log | Lets users log and track symptoms over time |
| 🧪 Result Summaries | Summarizes lab test and diagnostic results |
| 🩻 Triage Decisions | Helps prioritize cases based on risk indicators |
| 📝 Treatment Plans | Tracks recommended treatment steps |

## 🧠 AI Models Overview

- **Medical Image Analysis:** Deep learning model (Keras/H5) for breast cancer detection from medical images.
- **AI Chatbot:** LLM-powered conversational model for breast cancer awareness and support.

## 🛠️ Tech Stack

- C# / ASP.NET Core (Backend API)
- SQL Server with Entity Framework Core
- Python (AI models for image analysis and chatbot)
- Hangfire (background jobs & scheduled reminders)
- JWT (authentication & authorization)
- Swagger (API documentation)

AVIR was developed as a graduation project at the Faculty of Computers and Information, Fayoum Center - EELU. The goal is to support early breast cancer detection through accessible AI tools, encourage regular self-examination, and raise awareness through education and community support.
