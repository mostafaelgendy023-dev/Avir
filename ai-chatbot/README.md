# Breast Cancer Assistant | مساعد سرطان الثدي

A bilingual (Arabic/English) AI chatbot specialized in breast cancer information, powered by Groq (Llama 3.3 70B).

## Setup

Add your Groq API key in **Settings → Variables and Secrets**:

| Name | Value |
|------|-------|
| `GROQ_API_KEY` | `gsk_...` |

Get a free key from [console.groq.com](https://console.groq.com)
